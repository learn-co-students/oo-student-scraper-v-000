# learn-doctor

Check and fix your environment based on Learn.co's defaults.

## Installation

Install with:

```
$ gem install learn-doctor
```

Alternatively, add this line to your application's Gemfile:

```ruby
gem 'learn-doctor'
```

And then execute:

    $ bundle

## Usage

Run:

```
$ learn-doctor
```
